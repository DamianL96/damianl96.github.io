Proyectos

Biblioteca Virtual (C)

Conocimientos Aplicados: Estructuras Compuestas (Arboles binarios, lista de listas, filas), Punteros, Recursividad, Almacenamiento en archivos.

Este fue mi primer proyecto realizado en conjunto con otras personas, tomé el liderazgo del proyecto y a pesar de la poca experiencia del grupo (me incluyo) logré coordinar todas las partes para llegar a la fecha estimada de entrega, descartando funciones innecesarias y entregando un proyecto mas acotado a lo que se planificó pero muy solido.

Sistema para organizar y administrar una biblioteca, Los clientes pueden buscar distintos titulos y disponibilidad. El administrador puede realizar un ABM de los titulos, alquiler/devolucion o compra de libros para un cliente ingresado.

Tienda de Tecnología (Java)

Conocimientos Aplicados: POO, Genericidad, Interfaces, Excepciones, JSON, Consumo de APIs.

Segundo proyecto realizado con nuevos compañeros. Comenzamos con una idea demasiado ambiciosa que luego tuvimos que repensar y acotarla. A pesar de tener distintas opiniones y puntos de vista en el grupo, como equipo logramos dejar de lado las diferencias y encontrar un punto medio para poder llevarlo a cabo el trabajo de manera satisfactoria.

Sistema para gestionar una tienda virtual, Los clientes pueden buscar productos y comprarlos. El administrador puede realizar un ABM de productos y visualizar las facturas de compras efectuadas.

Simulador de batallas Pokemón (Angular)

Conocimientos Aplicados: Formularios reactivos, Guards, Consumo de APIs, Json-Server, Metodologia Scrum.

Proyecto mas reciente desarrollado con nuevos colegas. Por mi parte me encargue de realizar los formularios de registro y login, implementar los guards necesarios y el servidor que guarda los datos de las batallas. Mis colegas realizaron un complejo sistema de eleccion de personajes y batallas. Fue muy satisfactorio planificar los sprints, finalizar con exito cada uno de ellos y contar con colegas responsables que puedan resolver las tareas asignadas.

Simulacion de batalla Pokemón, Luego de iniciar sesion se ingresan los nombres de ambos jugadores, eligen personajes y comienza el combate, al finalizar se guardan los datos de la partida para visualizarlos luego.
